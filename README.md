# Pingoo Website

Landing page, Privacy Policy, Terms of Service, and Support page for the Pingoo app.

## Pages
- `/` — Landing page
- `/privacy.html` — Privacy Policy (required for App Store & Play Store)
- `/terms.html` — Terms of Service
- `/support.html` — Support & FAQ

## Deploy to GitHub Pages (Free)

1. Create a new GitHub repo named `pingoo-website`
2. Push this folder to that repo
3. Go to repo **Settings → Pages**
4. Set source to **main branch / root**
5. Your site will be live at: `https://yourusername.github.io/pingoo-website`

## URLs to use in App Store Connect / Google Play Console
- Privacy Policy: `https://yourusername.github.io/pingoo-website/privacy.html`
- Support URL: `https://yourusername.github.io/pingoo-website/support.html`
- Marketing URL: `https://yourusername.github.io/pingoo-website`
